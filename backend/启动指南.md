# 后端启动指南

## 前置条件
- ✅ JDK 17+ (已安装)
- ✅ MySQL 8.0 (已配置)
- ✅ Redis 7 (已配置)
- ❌ Maven (未安装)

## 启动方式

### 方式一：使用IDE启动（推荐）

1. 使用IntelliJ IDEA或Eclipse打开 `backend` 目录
2. 等待Maven自动下载依赖（首次需要几分钟）
3. 找到 `src/main/java/com/library/LibraryManagementSystemApplication.java`
4. 右键点击 -> Run 'LibraryManagementSystemApplication'

### 方式二：安装Maven后启动

1. 下载Maven：https://maven.apache.org/download.cgi
2. 解压到本地目录（如：`C:\apache-maven-3.9.5`）
3. 配置环境变量：
   - 新建系统变量 `MAVEN_HOME`，值为Maven解压路径
   - 编辑 `Path` 变量，添加 `%MAVEN_HOME%\bin`
4. 打开新的CMD窗口，运行：
   ```bash
   cd backend
   mvn spring-boot:run
   ```

### 方式三：使用Docker启动

1. 确保Docker Desktop已安装并运行
2. 执行：
   ```bash
   cd docker
   docker-compose up -d
   ```

## 验证启动成功

访问以下地址验证：
- 后端API：http://localhost:8080
- Swagger文档：http://localhost:8080/swagger-ui.html

## 数据库配置

确保MySQL和Redis正在运行，并且已执行数据库初始化脚本：

```bash
mysql -u root -p < backend/src/main/resources/schema.sql
```

默认账号：
- 管理员：admin / admin
- 普通用户：user / user

## 常见问题

### Q: 端口被占用
A: 修改 `backend/src/main/resources/application.yml` 中的端口号

### Q: 数据库连接失败
A: 检查MySQL是否启动，用户名密码是否正确

### Q: Redis连接失败
A: 检查Redis是否启动，端口是否为6379
